## Istruzioni per l'aggiornamento

Quando aggiungi un documento:
1. Copia il file nella cartella corretta secondo la struttura KB
2. Usa la convenzione di nome: `ANNO_TitoloBreve_[VolumeX].pdf`
3. Aggiungi una riga in questo registro con: file, titolo ufficiale, anno, note
4. Aggiorna il contatore totale
5. Segnala se il documento sostituisce o affianca una fonte precedente
